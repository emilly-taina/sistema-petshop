# Sistema Web para Pet Shop

Projeto final da disciplina  WEB I, com o objetivo de desenvolver o Front-End de um sistema web para um Pet Shop.

## Tema escolhido 

Sistema para Pet Shop.

## Funcionalidades

- Página inicial e identidade visual
- Serviços oferecidos pelo pet shop
- Agendamento de banho e tosa
- Cadastro de pets
- Área administrativa
